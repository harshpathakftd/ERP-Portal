# 🚀 ERP Portal (Django + DevOps)

A production-ready ERP (Enterprise Resource Planning) system built using Django, designed with a DevOps-first approach for scalability, automation, and cloud deployment.

---

## 📌 Features
- User Authentication & Role Management
- Modular ERP Architecture (Accounts, HR, Inventory, etc.)
- REST APIs for integration
- Secure and scalable backend
- Optimized database handling

---

## 🛠️ Tech Stack
- Backend: Django, Django REST Framework
- Database: PostgreSQL / MySQL
- Containerization: Docker
- CI/CD: Jenkins / GitHub Actions
- Code Quality: SonarQube
- Orchestration: Kubernetes (Minikube/K3s)
- Cloud: AWS (EC2, S3, RDS)

---

## ⚙️ DevOps Highlights
- Dockerized application (multi-stage builds)
- CI/CD pipeline for automated build, test & deployment
- Static code analysis using SonarQube
- Kubernetes deployment with scalable pods
- Environment-based configuration (dev/prod)
- Logging & monitoring ready setup
